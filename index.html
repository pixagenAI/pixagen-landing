Tutorial: Buat API Key Google AI

Langkah-langkah

  1. Buka Google AI Studio.
  2. Klik Create API key dan pilih project.
  3. Copy API key yang diberikan.
  4. Dalam Pixagen: Settings → API Key → paste.

Tip: Simpan key dalam password manager.
