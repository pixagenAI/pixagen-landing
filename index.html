Tutorial: Buat API Key Google AI

Langkah-langkah

  1. Buka Google AI Studio.
  2. Klik Create API key dan pilih project (atau create baru).
  3. Copy API key yang diberikan.
  4. Di dalam Pixagen, buka Settings → API Key dan paste key tersebut.

Tips: Simpan key dalam password manager. Jangan kongsi secara public.
